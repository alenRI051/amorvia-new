# Amorvia BETA 0.9.4 – Playtest Package

Dobrodošao u službeni **Amorvia 0.9.4 Playtest**.  
Ovaj paket je namijenjen internim i prijateljskim testerima koji žele pomoći da Amorvia postane stabilna, jasna i emocionalno korisna.

## Što testiramo u 0.9.4?

1. **Stabilnost enginea**  
   - učitavanje scenarija  
   - grananje (choices → sljedeći čvor)  
   - restart acta i promjena scenarija

2. **UI / HUD 9.8.6**  
   - čitljivost teksta  
   - vidljivost i ponašanje act badge-a  
   - izgled choice gumba  
   - trust / tension / childStress HUD

3. **Narativni flow**  
   - osjeća li se priča prirodno  
   - ima li konfuznih ili “prebrzih” prijelaza  
   - jesu li posljedice odluka razumljive

## Što je u ovom folderu?

- `Tester_Instructions.md` – kratke upute za testere  
- `Scenario_Coverage_Map.md` – koje scenarije i grane probati  
- `Bug_Report_Template.md` – kako prijaviti bug ili primjedbu  
- `Amorvia_Playtest_0.9.4_Package.zip` – ZIP paket (isti dokumenti + spremno za dijeljenje)

Ako imaš vremena za samo 20–30 minuta testiranja, to je već ogromna pomoć.  
Hvala! ❤️
