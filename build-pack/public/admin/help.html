Amorvia — Admin Help

How to list, export, prune logs.

Environment

  • TRACK_SALT — required
  • BLOB_READ_WRITE_TOKEN — required
  • ADMIN_TOKEN — recommended
  • TRACK_RATE_LIMIT — optional

Admin UI

Open /admin/. Set date → List Logs → Export CSV or JSONL.

If you set ADMIN_TOKEN, paste it in the “Admin Token” field.

Retention

Use the Retention controls to preview and prune logs older than N days.

// Dry run (no delete)
GET  /api/prune-logs?days=30&dryRun=1
// Delete (requires Admin Token header)
DELETE /api/prune-logs?days=30
Header: x-admin-token: <ADMIN_TOKEN>
