Amorvia BETA bundle

How to use
1) Upload all files to your site root (keep folders).
2) Ensure your server serves .js files with correct content-type (application/javascript).
3) Open / and pick a scenario from the Scenarios/Labs tabs.
4) Service worker is included (auto-refresh on new deploy).

Notes
- v2 scenarios live in /data/*.v2.json and are converted at runtime by /js/compat/v2-to-graph.js.
- The drop-in engine is /js/engine/scenarioEngine.js.
- Addons: Extras/Labs tabs, simple art loader.
